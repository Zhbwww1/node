#github使用
	1.使用git创建本地仓库
	2.在GitHub上创建github远程仓库
	3. 向远程仓库推送本地仓库 或 克隆远程仓库到本地仓库 				或 拉取远程仓库到本地仓库分支

#推送本地仓库到远程仓库github上（需要远程仓库账号密码）
	git push 远程仓库地址 本地仓库的分支名称
	git push 远程仓库地址别名 本地仓库的分支名称
	git push -u 远程仓库地址别名 本地仓库的分支名称
	(加上-u后缀会记住远程仓库地址和分支名称，下次推送时直接 git push)
	git remote add 远程仓库地址别名 远程仓库地址 （给远程仓库取别名）


#克隆远程仓库到本地仓库（无需账号密码）
	git clone 远程仓库地址	
	
	
#拉取远程仓库的指定分支到本地仓库
	git pull 远程仓库 远程仓库分支
